# Expense Tracker Web Dashboard

A responsive web dashboard for tracking personal expenses, built with React and TypeScript.

## 🛠️ Tech Stack
- React
- TypeScript
- Recharts (pie chart visualization)
- Axios (API communication)
- React Router DOM

## ✨ Features
- JWT Authentication (login & register)
- Add, view, and delete expenses
- Filter expenses by category
- Pie chart visualization by category
- Token refresh — automatic re-authentication
- Responsive design

## ⚙️ Getting Started
```bash
git clone https://github.com/plamenzubev/expense-tracker-web.git
cd expense-tracker-web
npm install
npm start
```

## 🌐 API
This app connects to the [Expense Tracker API](https://expense-tracker-api-w29w.onrender.com/api/).
